Shadowsocks 服务器安装

2017-10-25 16:53:31
other

MacOS 连接 远程服务器

sudo -i//切换到root

ssh root@ip //进行连接

搭建 Shadowsocks 服务

安装完成后配置服务器参数

$ vi /etc/shadowsocks.json

json 内容

配置防火墙

$ yum install firewalld
$ systemctl start firewalld

开启防火墙相应的端口

$ firewall-cmd --permanent --zone=public --add-port=443/tcp
$ firewall-cmd --reload

启动 Shadowsocks 服务

前台运行

$ ssserver -c /etc/shadowsocks.json

后台运行服务

sudo ssserver -c /etc/shadowsocks.json -d start
